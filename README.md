# E-COMMERCE NodeJS

Plataforma para ventas online desarrollada en el entorno de trabajo de Node.

### Instalacion

E-Commerce requiere [Node.js](https://nodejs.org/) v12+ para correr.

Instalar las dependencias y correr el servidor.

```sh
$ cd ecommerce-node
$ npm install 
$ npm start
```
